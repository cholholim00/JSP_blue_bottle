1. 카테고리 데이블

-------------------------------------------------------------------------------------------------
/    컬럼명    /             데이터 타입              /                    설명                       /
------------------------------------------------------------------------------------------------- 
/ category_id / INT (PK, AUTO_INCREMENT)  / 카테고리 고유번호 (기본키)            /
-------------------------------------------------------------------------------------------------
/ name         / VARCHAR(100)                    / 카테고리명 (예: 블렌드, 싱글오리진) /
-------------------------------------------------------------------------------------------------
/ description  / VARCHAR(255)                    / 카테고리 설명 (선택적)                   /
-------------------------------------------------------------------------------------------------

2.상품 테이블 [외래키 제약: products.category_id → categories.category_id ]

-------------------------------------------------------------------------------------------------
/    컬럼명    /             데이터 타입               /                    설명                       /
------------------------------------------------------------------------------------------------- 
/ product_id   / INT (PK, AUTO_INCREMENT)  / 상품 고유번호 (기본키)                  /
-------------------------------------------------------------------------------------------------
/ category_id  / INT (FK)                              / 카테고리 번호 (외래키)                  /
-------------------------------------------------------------------------------------------------
/ name         / VARCHAR(100)                     / 상품 이름                                    /
-------------------------------------------------------------------------------------------------
/ description  / VARCHAR(255)                    / 상품 설명                                     /
-------------------------------------------------------------------------------------------------
/ price          / DECIMAL(10,2)                     / 가격 (예: 15000.00)                        /
-------------------------------------------------------------------------------------------------
/ stock          / INT                                    / 재고 수량                                     /
-------------------------------------------------------------------------------------------------
/ created_at   / TIMESTAMP                        / 상품 등록 날짜                              /
-------------------------------------------------------------------------------------------------


3.상품 이미지 테이블 [외래키 제약: product_images.product_id → products.product_id ]

-------------------------------------------------------------------------------------------------
/    컬럼명     /             데이터 타입              /                    설명                        /
------------------------------------------------------------------------------------------------- 
/ image_id    /  INT (PK, AUTO_INCREMENT)  /        이미지 고유번호 (기본키)         /
-------------------------------------------------------------------------------------------------
/ product_id  /  VARCHAR(100)                    /           상품 번호 (외래키)               /
-------------------------------------------------------------------------------------------------
/ image_url   /  VARCHAR(255)                    /                 이미지 경로                   /
-------------------------------------------------------------------------------------------------
